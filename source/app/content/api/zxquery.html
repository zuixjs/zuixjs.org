### `ZxQuery` class *ZxQuery* is a class that implements a very lite subset of jQuery-like functionality. It can wrap elements of the DOM and provides useful methods for manipulating them. #### Constructor The constructor takes one optional parameter that can be a DOM element, a node list or any valid DOM query selector string expression. If no parameter is given, the ZxQuery will include the main `document` element. 
const zxElement = new ZxQuery(element);
or 
const zxElement = zuix.$(element);
which is the recommended way of creating `ZxQuery` objects.
